# `src/app` Folder

## Folder này để làm gì
Đặt phần khung ứng dụng: bootstrap, router, layouts, providers và global styles.

## Chứa loại file nào
- `main.tsx`, `App.tsx`
- `router.tsx`
- `layouts/*`
- `providers/*`
- `styles/*`

## Khi nào thêm file vào đây
Khi cần thay đổi app-level concern: route, global state provider, layout dùng chung, CSS toàn cục (bao gồm theme toàn trang).
