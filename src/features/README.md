# `src/features` Folder

## Folder này để làm gì
Chứa các khối UI/logic theo use-case nghiệp vụ, có thể tái dùng giữa nhiều page.

## Chứa loại file nào
- Components theo feature:
  - `discovery/*`
  - `booking/*`
  - `owner/*`
  - `admin/*`
- Bao gồm map discovery và các khối UI logic nghiệp vụ tái sử dụng.

## Khi nào thêm file vào đây
Khi một phần UI gắn với nghiệp vụ cụ thể và có khả năng dùng lại hơn một page.
