# `src` Folder

## Folder này để làm gì
Chứa toàn bộ mã nguồn frontend của NetUp MVP.

## Chứa loại file nào
- Khởi tạo app, router, layout.
- Pages theo từng role.
- Feature modules.
- Entity types/domain model.
- Mock data.
- Shared component + utility.

## Khi nào thêm file vào đây
Mọi file runtime của ứng dụng frontend (trừ config root như `vite.config.ts`, `package.json`) đều đặt trong `src`.
