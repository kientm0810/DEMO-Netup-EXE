# `src/shared` Folder

## Folder này để làm gì
Chứa phần dùng chung toàn app để tránh lặp code.

## Chứa loại file nào
- `components/*`: button, card, input, badge, empty state...
- `utils/*`: format currency/date, pricing, selectors, helper function.

## Khi nào thêm file vào đây
Khi một component hoặc hàm tiện ích có thể tái sử dụng cho nhiều feature/page khác nhau.
