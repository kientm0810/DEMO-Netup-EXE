# `src/pages` Folder

## Folder này để làm gì
Định nghĩa các màn hình theo route cụ thể.

## Chứa loại file nào
- Landing page, Not Found page.
- Page theo role:
  - `player/*`
  - `owner/*`
  - `admin/*`
  - `assistant/*` (chatbot demo)

## Khi nào thêm file vào đây
Khi có route mới hoặc cần tách một màn hình độc lập theo URL.
