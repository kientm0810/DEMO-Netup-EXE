# `src/entities` Folder

## Folder này để làm gì
Mô tả domain model và type definitions cốt lõi của hệ thống.

## Chứa loại file nào
- Types cho `player`, `court`, `session`, `booking`, `owner`, `admin`, `assessment`.
- Các interface/type export dùng xuyên suốt app.

## Khi nào thêm file vào đây
Khi nghiệp vụ có entity mới hoặc cần mở rộng thuộc tính cho entity hiện tại.
