# `src/mocks` Folder

## Folder này để làm gì
Quản lý toàn bộ mock data local cho demo MVP.

## Chứa loại file nào
- Dataset cho players, courts, sessions, bookings.
- Owner analytics và admin config mock.
- Player assessment và chatbot knowledge mock.

## Khi nào thêm file vào đây
Khi cần mở rộng dữ liệu demo hoặc tạo mock mới phục vụ luồng frontend mà chưa có backend.
